# Hermit <small>1.12</small>

> **无需Root**的Android自动化测试软件  

- 简单、轻便 (2.8MB)、无需Root
- 提供 restful API，支持所有主流语言  
- 已有基于其API封装的Python模块，使用Python轻松控制手机 
- 带有界面分析工具，可以分析界面层次和范围、获取界面上的控件信息
